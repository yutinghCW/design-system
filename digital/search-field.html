Search Fields 搜尋

搜尋（Search）讓使用者可以在此輸入文字快速尋找內容。

  • Design
  • Html

Usage

Search 讓使用者可以在此輸入文字快速尋找內容。

Types

Search 有兩種尺寸及兩種樣式。

Size Large Standard
Contained
Outlined

Anatomy

Search Fields 包含了 placeholder、搜尋 icon 及分隔內容與搜尋 icon 間的分隔線。

Class

Size Large Standard
Contained search__group--contained search__group--large search__group--contained
Outlined search__group--outlined search__group--large search__group--outlined

Html

<div class="search__group {{Classname}}">
  <input type="text" class="search__input" placeholder="請輸入關鍵字">
  <button type="submit" class="search__submit"><i class="icon icon-search"></i></button>
</div>
